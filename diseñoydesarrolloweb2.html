Diseño y Desarrollo Web

2do Año de Bachillerato - Creando el Internet del Mañana

El Stack Tecnológico

En este curso aprendemos que una web profesional nace de la combinación de tres pilares fundamentales. No solo escribimos código, sino que diseñamos experiencias de usuario (UX) atractivas y funcionales.

HTML5 (Estructura)
CSS3 (Estilo)
JavaScript (Lógica)

Diseño Responsivo

Uso de Media Queries para que la web se vea perfecta en tablets, PCs y celulares.

Interactividad

Creación de menús desplegables, sliders y validación de formularios con JS.

¿Qué aprenderás este año?

  • Multimedia Web: Inserción y optimización de audio, video y gráficos vectoriales (SVG).
  • Frameworks de Diseño: Introducción a herramientas como Bootstrap o Tailwind para acelerar el desarrollo.
  • Hosting y Dominios: Cómo subir tu proyecto a un servidor real para que todo el mundo lo vea.
  • Accesibilidad: Crear sitios que puedan ser utilizados por personas con discapacidades.

Ventajas y Desventajas

Ventajas

  • Creatividad Total: Puedes materializar cualquier idea visual que tengas.
  • Salida Laboral: Es una de las ramas con más empleo remoto en la actualidad.
  • Resultados Visibles: Ves el cambio en tiempo real mientras escribes el código.

Desventajas / Retos

  • Compatibilidad: Lo que se ve bien en Chrome puede verse mal en Safari o Firefox.
  • Curva de Aprendizaje: JavaScript puede ser frustrante cuando empiezas con la lógica avanzada.
  • Actualización: Cada mes salen herramientas nuevas que hay que aprender.
