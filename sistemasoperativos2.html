2do Año de Bachillerato - Administración de Servicios y Conectividad

Avanzando hacia la Administración

En segundo año, el estudiante deja de ser un usuario para convertirse en un **Administrador de Sistemas**. Se profundiza en la gestión de redes corporativas y en la seguridad de los datos mediante el uso de sistemas operativos de servidor como Windows Server y distribuciones avanzadas de Linux.

$ sudo apt-get update && sudo apt-get install apache2
# Configurando el servidor web...

Núcleos de Aprendizaje:

Servidores

Configuración de servicios DNS, DHCP y servidores web.

Subnetting

División lógica de redes IP para optimizar el tráfico.

Seguridad

Firewalls, permisos de usuario y políticas de grupo.

Redes de Área Local (LAN) Avanzadas

Estudiamos el diseño de red bajo estándares profesionales. Esto incluye el cableado estructurado y la configuración de dispositivos de red intermedios.

  • Switches y Routers: Diferencia funcional y configuración básica.
  • Estándares T568A/B: Creación de cables directos y cruzados con precisión técnica.
  • Topologías Lógicas: Cómo fluye la información más allá de la conexión física.

Ventajas y Desventajas

Ventajas

  • Perfil de Administrador: Capacidad para gestionar centros de cómputo pequeños y medianos.
  • Especialización: Base necesaria para certificaciones internacionales (como CCNA).
  • Seguridad: Aprendes a proteger redes contra accesos no autorizados.

Desventajas / Retos

  • Hardware Crítico: Los errores en servidores pueden afectar a muchos usuarios a la vez.
  • Matemática de Red: El cálculo de subredes (binario) requiere mucha práctica.
  • Actualización: Los protocolos de seguridad cambian constantemente.
